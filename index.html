📦 How to Buy a Bot

To buy a bot, first go to the support server and open a ticket for the bot you're buying, then wait for a support team member to give you the format.

Make sure to include:

  • The profile picture you want on the bot
  • The name you want for the bot
  • Any other customisations or features you want
  • Which tier you'd like (Bronze, Silver, or Diamond)

A team member will guide you through the rest of the process including payment and setup.

Open a Ticket →

🖥️ How to Become a Hoster

If you want to become a hoster you must meet the following requirements:

  • Ability to host 24/7 — if you need a day off, that's acceptable
  • Must be able to run code at 30–120ms latency
  • Must be able to leave your device open and running consistently
  • You cannot leak the code — doing so will result in legal consequences

You must be able to host on a device that can run code — PC, self-hosted servers, Raspberry Pi's, etc. You will be paid depending on how much you host. If you would like to be a hoster, please make a support ticket.
